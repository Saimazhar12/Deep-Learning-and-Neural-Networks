# 🐱🐶 Cats vs Dogs Image Classification using CNN

This project implements a deep learning-based image classification system to distinguish between cats and dogs using a Convolutional Neural Network (CNN). The model is trained on a dataset of 25,000 images and is capable of accurately classifying unseen images.

## 📌 Project Overview
The goal of this project is to build a binary classifier that identifies whether an image belongs to a cat or dog category. CNN is used to automatically learn important visual features from the images.

## 📂 Dataset
- Training Images: 20,000  
- Testing Images: 5,000  
- Classes: Cats 🐱, Dogs 🐶  

## ⚙️ Features
- CNN-based architecture  
- Data preprocessing and augmentation  
- Regularization using Dropout and Batch Normalization  
- Early stopping and model checkpointing  

## 🧠 Model & Training
- Optimizer: Adam  
- Loss Function: Binary Crossentropy  
- Batch Size: 32  
- Epochs: 20  

## 📊 Results
The model achieves strong performance on unseen test data, demonstrating good generalization ability.

## 🛠️ Technologies Used
- Python  
- TensorFlow / Keras  
- OpenCV  
- Matplotlib
